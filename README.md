# SQL Data Analysis Portfolio
A learning portfolio documenting my transition from media production to data analytics.


This repository contains SQL queries I wrote while learning data analysis on platforms like **W3Schools** and **SQL Academy**.  
The goal is to demonstrate my ability to write analytical SQL queries and extract business insights from data.

## 🍀 About this portfolio
After more than 20 years in television production, I am transitioning into data analytics.
My media background gave me strong skills in structuring information, understanding audiences,
and creating clear, data‑driven stories — all of which translate directly into analytical work.

This portfolio documents my progress in SQL and data analysis as I build the technical foundation
for my  Data Analyst roles in Canada.

## 📁 Files

- `project1_queries.sql` – a collection of practice queries covering:

## 🚀 How to use

2. Copy the query you want to run.
3. Paste it into the editor and execute.
4. See the results.
