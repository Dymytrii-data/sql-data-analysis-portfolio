# SQL Data Analysis Portfolio

This repository contains SQL queries I wrote while learning data analysis on platforms like **W3Schools** and **SQL Academy**.  
The goal is to demonstrate my ability to write analytical SQL queries and extract business insights from data.

## 📁 Files

- `project1_queries.sql` – a collection of practice queries covering:
  - Filtering and sorting (`SELECT`, `WHERE`, `ORDER BY`, `TOP`/`LIMIT`)
  - Aggregations (`COUNT`, `SUM`, `AVG`, `GROUP BY`, `HAVING`)
  - Joins (`INNER JOIN`, `LEFT JOIN`)
  - Conditional logic (`CASE`)
  - Subqueries and basic window functions (in progress)

## 🛠️ Skills demonstrated

- Writing SQL queries to answer business questions
- Joining multiple tables
- Grouping and aggregating data
- Creating customer segments using `CASE`
- Calculating revenue, order counts, and average values
- Documenting results and insights

## 🚀 How to use

1. Open any SQL environment (SQLite, MySQL, PostgreSQL, or the [W3Schools SQL Editor](https://www.w3schools.com/sql/trysql.asp?filename=trysql_select_all)).
2. Copy the query you want to run.
3. Paste it into the editor and execute.
4. See the results.
